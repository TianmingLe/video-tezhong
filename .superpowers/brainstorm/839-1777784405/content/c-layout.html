选择一个你更顺手的布局，用于“问题/关键词 → 4档相关性 → 主题聚类 → 导出/删除”。

输入区
结果区

Split View（推荐）

左侧输入与 run 选择，右侧实时展示“4档 + 主题簇 + 视频汇总”。适合反复改关键词快速迭代。

Tabs View

顶部是输入区，下面用 Tab 切换：概览 / 主题簇 / 知识点明细 / 视频明细。更整洁但切换更频繁。

Wizard

分步骤：选 runs → 填问题/关键词 → 运行 → 预览导出。更适合新手，但迭代调参更慢。
